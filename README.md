#  UPSTUDENT - Planner Acadêmico

O **UPSTUDENT** é um sistema de gerenciamento de grade horária universitária que permite ao aluno organizar sua semana de forma intuitiva, simulando a experiência de um caderno de estudos digital.

## 🚀 Tecnologias Utilizadas
- **React.js** com **Vite** para o core do projeto.
- **TypeScript** para tipagem estrita e segurança de dados.
- **Tailwind CSS** para estilização utilitária e responsiva.
- **JSON Server** para simulação de uma API REST.

## 🛠️ Como Rodar Localmente

1. **Clonar o repositório:**
   ```bash
   git clone https://github.com/victoriasantinni/UPStudent---Academic-Planer.git
Instalar as dependências:

Bash
npm install
Iniciar a Fake API (JSON Server):
(Em um terminal separado)

Bash
npx json-server --watch db.json --port 3000
Iniciar o projeto React:

Bash
npm run dev
Acesse: http://localhost:5173
